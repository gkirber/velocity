## ğŸš€ Getting Started

### Prerequisites

### Installation

1. Clone the repository:

```bash
git clone https://github.com/gkirber/velocity.git
cd velocity
```

2. Open `index.html` in your web browser or use a local server:

```bash

pnpm serve .
```

3. Navigate to `http://localhost:3000` in your browser
